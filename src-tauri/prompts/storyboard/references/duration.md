# Duration Guidelines

## Per-Panel Duration

| Complexity | Duration | Examples |
|------------|----------|----------|
| Simple hold / reaction | 2–3s | Character staring, still scene |
| Normal action or dialogue | 3–4s | Standard conversation, walking |
| Complex action or beat | 4–5s | Fight scene, important moment |
| Establishing / very long | 5–6s | Wide establishing shot |

## Rule of Thumb

- One breath ≈ 2 seconds
- Simple line of dialogue ≈ 2–3s
- Average panel ≈ 3–4s

## Panel Count Guidance

| Scene Length | Suggested Panels |
|-------------|------------------|
| Quick beat | 2–3 |
| Standard scene | 4–6 |
| Complex sequence | 6–8 |

## Batch Generation Rules

1. First panel establishes location (EWS or WS)
2. Vary shot types — avoid consecutive same shots
3. Build tension with progressively closer shots
4. End on a beat — each panel feels complete
5. Total duration should feel natural for the scene

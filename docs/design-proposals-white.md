# SceneForge White Mode Design Proposals

## Overview
4 distinct white mode design directions for SceneForge. Each represents a different aesthetic philosophy while maintaining functional consistency.

---

## Design A: Clean Studio
**Philosophy**: Minimal, warm, functional. Inspired by Linear/Notion.
- Background: `#FAFAF9` (warm white)
- Surface: `#FFFFFF`
- Border: `#E5E5E3`
- Text: `#1A1A1A` / `#6B6B6B`
- Accent: `#D97706` (amber-600, softer than original gold)

**Typography**: DM Sans throughout, Playfair Display for scene titles only.
**Feel**: Calming, professional, spacious.

---

## Design B: Editorial Warm
**Philosophy**: Warm, literary, editorial magazine feel.
- Background: `#F7F4F0` (warm cream)
- Surface: `#FFFEFA`
- Border: `#E0DCD4`
- Text: `#2C2416` (warm near-black)
- Accent: `#C4933F` (warm gold)

**Typography**: Playfair Display for all headers, DM Sans for body, DM Mono for script.
**Feel**: Sophisticated, warm, storytelling atmosphere.

---

## Design C: Modern Monochrome
**Philosophy**: High-contrast monochrome with sharp accents.
- Background: `#FFFFFF`
- Surface: `#F5F5F5`
- Border: `#D4D4D4`
- Text: `#000000` / `#525252`
- Accent: `#F59E0B` (bright amber)

**Typography**: DM Sans bold weights for hierarchy, DM Mono for metadata.
**Feel**: Bold, sharp, professional tool aesthetic.

---

## Design D: Soft Paper
**Philosophy**: Gentle, paper-like, inviting. Inspired by iA Writer/ Bear.
- Background: `#F9F7F4` (paper white)
- Surface: `#FFFFFF`
- Border: `#EAE7E1`
- Text: `#3D3A36` (soft charcoal)
- Accent: `#D4A853` (muted gold)

**Typography**: DM Sans light/regular, generous line-height, generous spacing.
**Feel**: Calm, focused, distraction-free.

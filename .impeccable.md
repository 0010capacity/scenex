# SceneX Design Context

## Users

**Primary**: 유튜버, 콘텐츠 크리에이터
- 개인 또는 소규모 팀으로 영상 제작
- 시간 제약이 있어 빠른 워크플로우 필요
- 전문 스토리보드 툴보다 접근 가능한 도구 선호
- AI 활용에 열려 있음

**Context**:
- 촬영 전 시나리오 구상 및 스토리보드 작성
- 팀원과의 컨셉 공유
- 클라이언트 프레젠테이션

## Brand Personality

**3 Words**: Professional · Minimal · Precise

**Voice & Tone**:
- 자신감 있지만 과시하지 않음
- 복잡함보다 명확함
- 크리에이터의 창의성을 방해하지 않고 돕는 도구
- "less is more" 철학

**Emotional Goals**:
- 신뢰감: 안정적으로 작동하는 전문 도구
- 집중: 불필요한 요소 없이 작업에 몰입
- 효율: 최소 클릭으로 최대 결과

## Aesthetic Direction

**Visual Tone**: Clean Studio
- Figma, Linear, Notion 같은 현대적 생산성 툴 레퍼런스
- 다크 모드 기본, 라이트 모드 지원
- 충분한 여백과 명확한 계층 구조

**References**:
- Figma: 패널 구조, 속성 인스펙터
- Linear: 미니멀한 다크 테마, 섬세한 인터랙션
- Notion: 문서 중심 레이아웃, 깔끔한 타이포그래피

**Anti-references**:
- Adobe 제품의 복잡한 툴바/패널
- 구식 스토리보드 툴의 클로키한 UI
- 과도한 그라디언트/글로우 효과

**Theme**:
- Dark mode: 딥 블랙 (#0B0C10) + 골드 악센트 (#E8A838)
- Light mode: 크림 화이트 (#FAFAF9) + 인디고 악센트 (#4F46E5)
- 배경은 레이어드 (bg0 → bg4 깊이감)
- 보더는 미세하게, 섀도우는 절제

## Design Principles

1. **Tool, Not Toy**
   장식보다 기능. 모든 시각 요소는 목적이 있어야 함.

2. **Calm Focus**
   크리에이터의 아이디어가 주인공. UI는 조용히 뒤에서 지원.

3. **Precision in Details**
   1px의 차이가 만드는 정제된 느낌. 스페이싱, 정렬, 타이포그래피 일관성.

4. **Progressive Disclosure**
   기본은 심플하게. 고급 기능은 필요할 때만 노출.

5. **Motion with Purpose**
   애니메이션은 상태 변화를 설명할 때만. 불필요한 모션 금지.

## Typography

**Font Stack**:
- Sans: DM Sans — UI 텍스트, 버튼, 라벨
- Mono: DM Mono — 숫자, 코드, 슬러그라인
- Serif: Playfair Display — 타이틀, 인용 (제한적 사용)

**Scale**:
- Base: 14px (body)
- Small: 10-12px (labels, captions)
- Large: 15-16px (headers)

## Color Usage

**Dark Mode**:
- Background layers: bg0 → bg4 (깊이감)
- Primary accent: Gold (#E8A838) — AI 관련, 하이라이트
- Secondary accents: Blue, Green, Red, Purple — 상태 표시
- Text: #E8E5DC (primary), #9A9790 (secondary), #7A7872 (tertiary)

**Light Mode**:
- Background layers: #FAFAF9 → #E5E5E3
- Primary accent: Indigo (#4F46E5)
- Text: #1A1A1A (primary), #6B6B6B (secondary)

## Accessibility

- 기본 웹 접근성 준수
- 충분한 색 대비 (WCAG AA 수준)
- 키보드 내비게이션 지원
- prefers-reduced-motion 존중
- 특별한 색약 지원 모드는 불필요
